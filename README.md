# Alumni EI's website

## Requirements

- Ruby 2.4.2
- Jekyll 3.6.2

## Development

1. `git clone git@github.com:alumniei/website.git`
2. `cd website`
3. `bundle install`
4. `jekyll serve`
5. go have some fun

### Deployment
1. From the project root directory `jekyll build`
2. Copy the contents of folder `_site` to your web server and that's all you need
